\# Deployment Guide



\## Overview



This document explains how the DevOps Telegram Monitoring Bot can be deployed on a Linux server using Docker and Docker Compose.



Actual VPS deployment is optional. The project is currently portfolio-ready and can be run locally with Docker Desktop or on any Linux server with Docker installed.



\---



\## Requirements



A Linux server or local Linux environment with:



\* Ubuntu 22.04 or 24.04

\* Git

\* Docker

\* Docker Compose

\* Telegram bot token

\* Telegram chat ID
